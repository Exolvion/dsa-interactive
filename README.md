# DSA Interactive — Lectures 6-10

Interactive, visually-driven study site for INF1008 Data Structures & Algorithms, covering:

- **Lecture 6** — Search Algorithms 1: Linear Search, Binary Search, Hash Tables (chaining, linear/quadratic probing, double hashing)
- **Lecture 7** — Search Algorithms 2: Binary Search Trees, 2-3 Trees, Left-Leaning Red-Black Trees
- **Lecture 8** — Graph Algorithms 1: DFS, BFS, Tree Traversals, Topological Sort
- **Lecture 9** — Graph Algorithms 2: Prim, Kruskal, Dijkstra, Bellman-Ford
- **Lecture 10** — String Algorithms: Brute-Force Search, Knuth-Morris-Pratt (KMP)

Live site: https://exolvion.github.io/dsa-interactive/

## Development

Static site, no build step. Open `index.html` directly, or serve locally:

```
npx serve .
```
