# Supplementary Sources

The INF1008 lecture content (terminology, worked examples like `tinyEWG`, LLRBT,
symbol tables) closely follows Sedgewick & Wayne's *Algorithms* (Princeton COS226).
These are the sources used to sharpen intuition text and linked as "Further reading"
on each lecture page. Not reproduced verbatim — used for reference only.

## Lecture 6 — Search Algorithms 1 (linear/binary search, hashing)
- Princeton algs4 — Symbol Tables: https://algs4.cs.princeton.edu/31elementary/
- Princeton algs4 — Hash Tables: https://algs4.cs.princeton.edu/34hash/

## Lecture 7 — Search Algorithms 2 (BST, 2-3 tree, LLRBT)
- Princeton algs4 — Binary Search Trees: https://algs4.cs.princeton.edu/32bst/
- Princeton algs4 — Balanced Search Trees: https://algs4.cs.princeton.edu/33balanced/
- USF interactive visualizations (BST / Red-Black / 2-3-4 tree): https://www.cs.usfca.edu/~galles/visualization/

## Lecture 8 — Graph Algorithms 1 (DFS, BFS, topological sort)
